# Inter Font Files

This directory should contain the Inter font files for the Corpfinity Employee App.

## Required Font Files

The following Inter font files are required:
- `Inter-Regular.ttf` (weight: 400)
- `Inter-Medium.ttf` (weight: 500)
- `Inter-SemiBold.ttf` (weight: 600)
- `Inter-Bold.ttf` (weight: 700)

## Download Instructions

1. Visit the official Inter font repository: https://github.com/rsms/inter
2. Download the latest release from: https://github.com/rsms/inter/releases
3. Extract the font files from the `Inter Desktop` folder
4. Copy the required `.ttf` files to this directory

Alternatively, you can download Inter from Google Fonts:
https://fonts.google.com/specimen/Inter

## Note

For development purposes, if the Inter font files are not available, Flutter will fall back to the system default font (Roboto on Android, San Francisco on iOS).

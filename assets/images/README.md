# Images Directory

## Required Logo File

Please save the Corpfinity logo as:
- **File name**: `corpfinity_logo.png`
- **Location**: This directory (`assets/images/`)
- **Format**: PNG with transparent background (recommended)
- **Size**: 512x512 pixels or higher for best quality

## Current Status

⚠️ **Action Required**: Save the logo image shared in the chat as `corpfinity_logo.png` in this directory.

The logo will be displayed in:
- Splash screen (app launch)
- Sign-in screen
- Sign-up screen
- Any other location where you use the `AppLogo` widget

## Fallback

If the logo file is not found, the app will display a fallback spa icon in turquoise color.
